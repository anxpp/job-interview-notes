#### 字符类型

    char 以 32 位值表示单个 Unicode 字符

需要转义的字符： '\''  '\\'  '\n'  '\r'  '\t'

显示转换：

```
assert_eq!('*' as i32, 42);
assert_eq!('*' as u16, 0x2a);
assert_eq!('*' as u8, 42); // 高位会被截断
```

反向转换只能： u8 as char
